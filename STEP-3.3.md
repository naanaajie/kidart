## 改了什么
- **社交 Tab**：免费导出，自动给图右下角加 **stylejoy** 水印。
- **打印 / 自定义 Tab**：付费高清，点导出时走门禁：
  - 没登录 → 弹登录框
  - 登录了但没权限（非 lifetime 且 0 次）→ 弹**付费弹窗**（3.3 是占位，3.4 接 PayPal）
  - 有权限 → 先渲好图 → 服务端 `spend_single_credit` 扣额度 → 成功才下载、记流水、刷新额度
- 价格从数据库 `pricing` 表读，自动显示 $3.90 / $19.90。
